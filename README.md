# 開発経験まとめ
## ESPI(電子スペックル干渉法）
### 2025年6月～2025年8月
Whitman Collegeの物理工学部、Kurt Hoffman教授のもと、ESPI（電子スペックル干渉法）の研究を行いました。研究室設備の最適化、そしてヴィオラやピアノなどの楽器の振動解析をナノメートル単位で行いました。
<img width="910" height="403" alt="スクリーンショット 2026-03-23 052709" src="https://github.com/user-attachments/assets/4a9519c5-c9cd-4c66-b2af-630696819211" />
振動のモデリングをCOMSOLで行った後、短時間でのレーザーの位相シフトを現像化することで、観測する物体の振動モードの形の特定と、共鳴周波数の測定を成功させました。この技術はナノメートル単位かつ非接触・非破壊でで物体の振動の特徴を可視化することができるため、物体を傷つける心配を最小に抑えたまま品質検査を行えるという利点があります。<img width="876" height="600" alt="Exp 125 Piezo 4 5 Speak 5 f440 000" src="https://github.com/user-attachments/assets/9c280c18-da46-4609-be26-3eee866b4b4f" />
